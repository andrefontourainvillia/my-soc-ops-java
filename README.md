## What is Soc Ops?

Soc Ops is a **Social Bingo icebreaker game** designed for in-person events, workshops, and team mixers. Each player gets a randomised 5Г—5 board filled with prompts like *"Has lived in another country"* or *"Owns a houseplant"*. Walk the room, find people who match the squares, and be the first to fill a row, column, or diagonal.

This repository is also a **hands-on GitHub Copilot Agent Lab** вҖ” you build the game while mastering VS Code's Agent Mode, custom agents, and context engineering.

## рҹҡҖ Quick Start

**Prerequisites:** [Java 21 JDK](https://adoptium.net/) В· [Maven 3.9+](https://maven.apache.org/) (or use the included wrapper)

```bash
# Clone and run
git clone <your-repo-url>
cd my-soc-ops-java/socops
./mvnw spring-boot:run
```

Open [http://localhost:8080](http://localhost:8080) вҖ” your bingo board is ready to play!

```bash
# Build & test
./mvnw clean package   # compile + package
./mvnw test            # run all unit tests
```
